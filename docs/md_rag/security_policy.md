# 入館管理とアクセス制御

施設への入館には必ず会員証またはビジターカードが必要です。入口では警備員による本人確認を行います。会員証を忘れた場合は、受付にて身分証明書を提示し、臨時カードの発行を受けてください。

入館時と退館時には必ず受付でカードをタッチしてください。これにより施設内の在館者を把握し、緊急時の安全確保に役立てています。
---
# カメラとモニタリング

施設内には防犯カメラが設置されており、24時間監視を行っています。録画データは30日間保存され、セキュリティインシデント発生時の調査に使用されます。

カメラの設置場所は入口、廊下、共用エリアです。個室内やトイレにはカメラは設置されていません。プライバシーに配慮した運用を心がけています。
---
# 緊急時の対応

火災や地震などの緊急事態が発生した場合は、館内放送の指示に従って避難してください。避難経路は各フロアの見やすい場所に掲示されています。

緊急連絡先は以下の通りです：
- 警備室: 内線999
- 消防署: 119
- 警察署: 110

避難訓練は年2回実施されます。利用者の皆様のご協力をお願いいたします。
