# 施設利用について

当施設は月曜日から金曜日まで、午前9時から午後6時まで利用可能です。土日祝日は休館となります。利用には事前の予約が必要で、予約は利用日の3営業日前までに行ってください。

予約方法は、オンライン予約システムまたは電話での予約が可能です。キャンセルは利用日の前日までに行ってください。

# 利用料金

施設の利用料金は以下の通りです：
- 1時間あたり: 1,000円
- 半日（4時間）: 3,500円
- 1日（8時間）: 6,000円

学生や65歳以上の方は、証明書の提示により30%の割引が適用されます。また、月額会員制度もあり、月額10,000円で1日2時間まで毎日利用可能です。

支払い方法は、現金、クレジットカード、電子マネーに対応しています。

# 注意事項とルール

施設内では以下のルールを遵守してください：
1. 施設内は禁煙です
2. 飲食は指定エリアのみで可能です
3. 携帯電話はマナーモードに設定してください
4. 貴重品は各自で管理してください

ルール違反があった場合、利用停止となる可能性があります。また、施設の設備を破損した場合は、修理費用を請求させていただきます。
