# FTS処理用プロンプト

以下のマークダウンファイルをFTS検索用に処理してください。

## 処理要件

1. **フロントマターYAML（1ブロック目）**: そのまま保持してください

2. **本文ブロック（2ブロック目以降）**: 
   - 元のコンテンツはそのまま維持
   - 各ブロック（`---`で区切られたセクション）の末尾に`[FTS]...[/FTS]`タグを追加
   - タグ内には検索用キーワードをスペース区切りで記述

3. **[FTS]タグ内のキーワード選定基準**:
   - 本文に含まれる重要な単語・数値・固有名詞
   - 検索時に使われそうな類義語や関連語
   - ユーザーが検索する際に使いそうな表現
   - 日時、金額、場所などの具体的な情報
   - 略語や正式名称の両方

4. **出力形式**:
```
---
(フロントマターYAMLをそのまま)
---
# セクションタイトル

(本文をそのまま)

[FTS]キーワード1 キーワード2 キーワード3 ...[/FTS]

---
# 次のセクションタイトル

(本文をそのまま)

[FTS]キーワード1 キーワード2 ...[/FTS]
```

## 処理対象ファイル

## 使用例

**入力:**
```markdown
---
document_name: "会議室予約ガイド"
category: "施設"
---
# 予約方法

会議室の予約はWebシステムから行えます。利用日の7日前から予約可能です。
```

**出力:**
```markdown
---
document_name: "会議室予約ガイド"
category: "施設"
---
# 予約方法

会議室の予約はWebシステムから行えます。利用日の7日前から予約可能です。

[FTS]予約方法 会議室 予約 Webシステム オンライン 利用日 7日前 1週間前[/FTS]
```
